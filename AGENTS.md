# CandyLauncher

## Build Commands

> No reading required @CMakePresets.json

### Generate, CMake with Ninja + MSVC

```
cmd.exe /c .\scripts\run_cmake_generate.cmd
```

### Build

```
cmd.exe /c .\scripts\run_build.cmd
```

### The command will cause the program to stop itself and output debugging logs after 3 seconds

```
cmd.exe /c .\scripts\run_candylauncher.cmd
```

### Test Commands
```
cmd.exe /c .\scripts\run_debug_test.cmd
```
