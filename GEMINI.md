# GEMINI.md

This file provides guidance to gemini (google.com/Gemini) when working with code in this repository.

## Build Commands

> IMPORTANT: No need to read any files, just execute the following command

### Generate, CMake with Ninja + MSVC

```shell
./scripts/run_cmake_generate.cmd
```

### Build

```shell
./scripts/run_build.cmd
```

### The command will cause the program to stop itself and output debugging logs after 3 seconds

```bash
./scripts/run_candylauncher.cmd
```

### Test Commands
```shell
./scripts/run_debug_test.cmd
```
