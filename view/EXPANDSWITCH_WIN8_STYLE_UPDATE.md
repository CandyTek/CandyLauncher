# ExpandSwitch 控件样式更新 - 采用Win8风格

## 变更概述

将 `expandswitch` 控件中的开关样式从之前的GDI+圆角设计更改为复用 `SwitchPaintWin8` 的Windows 8.1扁平风格,以保持整个应用程序UI的一致性。

## 主要变更

### 1. 视觉样式改变

**之前 (GDI+圆角风格):**

- 圆角矩形轨道
- 圆形滑块
- 尺寸: 40x20px
- 使用GDI+抗锯齿绘制

**现在 (Win8扁平风格):**

- 直角矩形轨道
- 矩形滑块(12x20px)
- 尺寸: 46x18px
- 使用纯GDI绘制
- 2px边框
- 黑色滑块

### 2. 代码变更

**文件**: `view/CustomButtonHelper.hpp`

#### DrawMiniSwitch 函数重写

- 移除了GDI+相关代码
- 采用与 `SwitchPaintWin8` 相同的绘制逻辑
- 使用纯GDI API (Rectangle, FillRect等)
- 不再需要Gdiplus头文件

#### GetSwitchRect 函数更新

- 开关宽度: 40 → 46
- 开关高度: 20 → 18

#### 移除依赖

```cpp
// 已移除
#include <gdiplus.h>
#pragma comment(lib, "gdiplus.lib")
```

### 3. 样式细节对比

| 特性    | 之前               | 现在               |
|-------|------------------|------------------|
| 轨道形状  | 圆角矩形             | 直角矩形             |
| 滑块形状  | 圆形               | 矩形               |
| 滑块颜色  | 白色               | 黑色               |
| 边框    | 无                | 2px灰色            |
| 关闭轨道色 | RGB(200,200,200) | RGB(173,173,173) |
| 开启轨道色 | RGB(0,120,215)   | RGB(0,120,215)   |
| 悬停关闭色 | RGB(180,180,180) | RGB(203,203,203) |
| 悬停开启色 | RGB(56,165,250)  | RGB(56,165,250)  |
| 绘制技术  | GDI+             | GDI              |

## 优势

1. **UI一致性**: 与主程序设置窗口中的其他开关控件保持一致的视觉风格
2. **性能提升**: 纯GDI绘制比GDI+更轻量
3. **代码复用**: 直接复用已有的`SwitchPaintWin8`样式逻辑
4. **减少依赖**: 不再需要GDI+库
5. **扁平化设计**: 符合Windows 8/10的现代UI设计语言
